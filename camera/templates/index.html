Face Recognition Training

I am trying to make a face recognition program to recognise everyone's faces in the Isreal tour photos.
Please help by teaching the program everyone's names.

On every page an image will be shown at the top of the page.
Below there may be 1 or more cut out faces. Some will have the correct names next to the while some may not.
If the correct name shown, just continue.
If the name is incorrect or "Unknown", correct it.
If you are not sure or want to skip that one, leave the input box blank.
If there is no face in the picture enter "none".

As the program is looking through the high quality photos for faces on each page load, it can take as much as 30 seconds for the page to load.
